# binary_progress_bar
A progress bar demo with "binary" display.

![img](./img.png)

### Usage
You can start a local server directly in Python

```bash
$ python -m http.server 80
```

Then open in your browser. [http://127.0.0.1:80/](http://127.0.0.1:80/)
